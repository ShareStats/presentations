Part 1

Introduction to ShareStats

What is ShareStats?

ShareStats is an open-source item bank for statistics education.

Goals

  • Lower the barrier to writing good statistics exams
  • Promote reuse of high-quality questions
  • Support both formative and summative assessment

Key features

  • Freely available questions (open licence)
  • Items in multiple formats
  • Compatible with ANS/Canvas platforms
  • Powered by the R exams package

Why an item bank?

Writing good statistics questions is time-consuming.

ShareStats helps educators with question creation.

  • Peer-reviewed items written by statistics educators
  • Covers common introductory statistics topics
  • Items are tagged by topic, difficulty, and format

What’s in the bank?

Topics covered include:

Category Example
Measurement level Nominal vs ordinal vs interval (e.g., survey gender = nominal)
Inferential statistics Two-sample t-test to compare group means
Probability Compute P(X ≤ 2) for X ~ Poisson(λ=1.5)
Assumptions Check normality of residuals in linear regression
Descriptive statistics Mean, median, and standard deviation of scores
Variable type Continuous (age), discrete (count), categorical (gender)
Distributions Normal, binomial, Poisson (e.g., heights ≈ Normal)
Factor analysis Exploratory factor analysis on questionnaire items
Reliability Cronbach’s alpha for internal consistency

Item formats

ShareStats items are written in R/exams format, supporting:

  • Single-choice (schoice)
  • Multiple-choice (mchoice)
  • Numeric answer (num)
  • Open answer (string)
  • Cloze (combination of the above)

Each item is randomisable — different numbers, datasets, or scenarios per student.

Part 2

Searching & Uploading to ANS

Finding items on ShareStats

  1. Go to sharestats.nl
  2. Browse by topic or use the search function
  3. Preview the item — question text, answer options, solution
  4. Note the item ID or download the .Rmd file directly

Downloading items with R exams

Install and load the package:

install.packages("exams")
library(exams)

Download an item from the ShareStats GitHub repository:

# Example: download a t-test item
url <- "https://raw.githubusercontent.com/ShareStats/ShareStats/main/items/ttest_onesample.Rmd"
download.file(url, destfile = "ttest_onesample.Rmd")

Compiling a set of items

Combine multiple items and render to a PDF exam or QTI (for ANS/Canvas):

library(exams)

# List the items you want
my_items <- c(
  "ttest_onesample.Rmd",
  "descriptives_mean.Rmd",
  "regression_interpret.Rmd"
)

# Render to QTI 2.1 format (for ANS)
exams2qti21(
  file      = my_items,
  n         = 1,          # number of exam versions
  dir       = "output",
  name      = "stats_quiz"
)

Uploading to ANS

Once you have the .zip QTI file:

  1. Log in to ANS (ans.app)
  2. Open your course → Item bank → Import
  3. Choose QTI 2.1 and upload the .zip
  4. ANS parses the items — preview and assign to an assignment
  5. Set feedback, scoring, and release settings as usual

Note

The same workflow applies to Canvas using exams2canvas() and QTI 1.2.

Uploading to Canvas

For Canvas, export to QTI 1.2 instead:

exams2canvas(
  file  = my_items,
  n     = 1,
  dir   = "output",
  name  = "stats_quiz"
)

Then import via Canvas → Settings → Import Course Content → QTI.

Part 3

Your Turn — Hands-On Practice

Your turn!

Let’s try it together

Exercise 1 — Browse & Pick

  1. Open sharestats.nl on your laptop
  2. Find two items on a topic relevant to your course
  3. Download the .Rmd files to a local folder

Tip: look at the item source — all items are plain R Markdown files.

Exercise 2 — Compile a Mini Exam

With the items you downloaded, compile a QTI file:

library(exams)

my_items <- c("item1.Rmd", "item2.Rmd")  # your downloaded items

exams2qti21(
  file = my_items,
  n    = 1,
  dir  = "output",
  name = "my_quiz"
)

Check the output/ folder — you should see a .zip file ready to upload.

Exercise 3 — Upload to ANS or Canvas

  • ANS
  • Canvas
  1. Log in to ANS → your course
  2. Item bank → Import → QTI 2.1
  3. Upload the .zip from output/
  4. Preview one item
  1. Log in to Canvas → your course
  2. Settings → Import Course Content
  3. Select QTI → upload the .zip
  4. Find the items in your Question Banks

Common questions

Issue Solution
Item doesn’t render Check R version ≥ 4.0 and tinytex installed
QTI import fails Ensure you’re using QTI 2.1 for ANS
Images missing Use exams2qti21(..., edir = "images")
Reproducibility Set set.seed() before exams2qti21()
